MV_EXPANDedit

The MV_EXPAND processing command expands multivalued fields into one row per value, duplicating other fields:

ROW a=[1,2,3], b="b", j=["a","b"]
| MV_EXPAND a
a:integer b:keyword j:keyword

1

b

["a", "b"]

2

b

["a", "b"]

3

b

["a", "b"]
