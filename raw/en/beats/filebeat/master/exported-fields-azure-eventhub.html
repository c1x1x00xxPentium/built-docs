Fields regarding the Azure event hub information.

azure-eventhubedit

Fields regarding the Azure event hub information.

azure-eventhub.offset

The position of an event within a partition.

type: long

azure-eventhub.sequence_number

The logical sequence number of the event within the partition stream of the event hub.

type: long

azure-eventhub.enqueued_time

The date and time, in UTC, of when the event was enqueued in the event hub partition.

type: date

azure-eventhub.eventhub

The name of the event hub instance.

type: keyword

azure-eventhub.consumer_group

The view (state, position, or offset) of an entire event hub.

type: keyword
