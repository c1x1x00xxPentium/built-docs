Virtual network traffic filtersedit

Traffic filtering, to only Azure Virtual Network connections, is one of the security layers available in Elastic Cloud Enterprise. It allows you to limit how your deployments can be accessed.

Read more about Traffic Filtering for the general concepts behind traffic filtering in Elastic Cloud Enterprise.

Virtual network filtering is supported only for Azure regions.
