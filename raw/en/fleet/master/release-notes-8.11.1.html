Fleet and Elastic Agent 8.11.1edit

Review important information about Fleet Server and Elastic Agent for the 8.11.1 release.

Breaking changesedit

  • The breaking change that could prevent the Elastic Agent or Integrations Server component from booting up within an ECE deployment has been resolved in this release.

Known issuesedit

Version 8.11.0 is not in the list of Elastic Agent versions in Kibana Fleet UI

Details

On the Fleet UI in Kibana version 8.11.0:

  • When adding a new Elastic Agent, the user interface shows version 8.10.4 instead of 8.11.0.
  • When attempting to upgrade, the modal window to pick the version shows 8.10.4 as latest version.

Impact

For a workaround, refer to the steps in this knowledge article: Cannot upgrade Fleet Managed Elastic Agents to 8.11.0 via Fleet UI in 8.11.0.

New featuresedit

The 8.11.1 release Added the following new and notable features.

Elastic Agent
  • Add the dimensions component.id and component.binary to Elastic Agent and Beats monitoring output, to support unique entries for the Time Series Database (TSDB) feature. #3626 #7977

Bug fixesedit

Fleet
  • Append space ID to security solution tag. (#170789).
  • Modify bulk unenroll to include inactive agents. (#170249).
