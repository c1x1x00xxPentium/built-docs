Virtual network traffic filtersedit

Traffic filtering, to only Azure Virtual Network connections, is one of the security layers available in Elasticsearch Service. It allows you to limit how your deployments can be accessed.

Read more about Traffic Filtering for the general concepts behind traffic filtering in Elasticsearch Service.

Virtual network filtering is supported only for Azure regions.
