Elasticsearch Hadoop breaking changesedit

This list summarizes the most important breaking changes in Elasticsearch Hadoop 8.0.0. For the complete list, go to Elasticsearch Hadoop breaking changes.

The default scroll size is now 1000edit

We’ve increased the es.scroll.size default from 50 to 1000. If you deploy ES-hadoop in a low memory environment, consider lowering es.scroll.size to avoid issues.

Support for Scala 2.10 has been removededit

Our compatibility artifacts no longer support Scala 2.10. Spark deprecated support for Scala 2.10 in the Spark 2.0.0 release.
